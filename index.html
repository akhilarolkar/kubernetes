⭐ Classic interview question

"When replicas change from 2 to 4, who actually creates the new pods?"

Answer: The ReplicaSet Controller inside kube-controller-manager on the Masternode.

It runs a reconcile loop: desired = 4, actual = 2 → create 2 pod objects. The scheduler does NOT create pods. The API server does NOT create pods. The kubelet on the workernode does NOT create pods. Only the ReplicaSet Controller creates the pod API objects.
